# Agent Notes

This project is a small Next.js App Router prototype for a solo Sangoku card game.

## Local Commands

- Install dependencies: `npm install`
- Start development server: `npm run dev`
- Lint: `npm run lint`
- Test: `npm test`

## Project Shape

- `app/` contains App Router pages.
- `components/` contains game UI components.
- `lib/game/` contains pure game data and engine logic.
- `tests/` contains Vitest tests for the pure engine.
