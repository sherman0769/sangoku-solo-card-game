# 三國單騎傳

《三國單騎傳》是一款單人三國卡牌闖關遊戲。玩家選擇武將，使用斬、閃、酒、兵書與破甲連闖三關，擊敗最終 Boss 呂布。

線上遊玩網址：[https://sangoku-solo-card-game.vercel.app](https://sangoku-solo-card-game.vercel.app)

目前版本：`v0.3.0 三武將測試版`

## 已完成功能

- 首頁、遊戲頁與勝敗結果頁
- 武將選擇：關羽、趙雲、諸葛亮
- 關羽技能「武聖」：每回合第一次使用斬時傷害 +1
- 趙雲技能「龍膽」：閃可當斬、斬可當閃
- 諸葛亮技能「觀星」：每回合開始時先查看牌堆頂最多 3 張，選 1 張加入手牌，其餘放回牌堆底，再正常抽 2 張牌
- 武將定位：關羽「攻擊爆發」、趙雲「攻防靈活」、諸葛亮「策略控牌」
- 三關敵人流程：黃巾兵、山賊頭目、呂布
- 玩家手牌與士氣系統
- 五種卡牌：斬、閃、酒、兵書、破甲
- 敵人行動：普通攻擊、猛攻、防守、蓄力
- 敵人攻擊時可選擇使用閃或承受傷害
- 戰後獎勵三選一與玩家成長
- 戰鬥紀錄、關卡提示、操作回饋與輕量動畫
- 手機與桌機版基本響應式排版

## 本地啟動

安裝依賴：

```sh
npm install
```

啟動開發伺服器：

```sh
npm run dev
```

開啟 [http://localhost:3000](http://localhost:3000)。

## 測試方式

執行 ESLint：

```sh
npm run lint
```

執行 Vitest：

```sh
npm test
```

執行 production build：

```sh
npm run build
```
