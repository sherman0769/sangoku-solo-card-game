# 三國單騎傳

《三國單騎傳》是一款單人三國卡牌闖關遊戲。玩家選擇武將，使用斬、閃、酒、兵書、破甲、裝備與戰術卡挑戰第一章 8 關，擊敗最終 Boss 呂布。

線上遊玩網址：[https://sangoku-solo-card-game.vercel.app](https://sangoku-solo-card-game.vercel.app)

目前版本：`v0.13.3 開場動畫體驗優化版`

## 已完成功能

- 首頁、遊戲頁與勝敗結果頁
- 武將選擇：關羽、趙雲、諸葛亮
- 關羽技能「武聖」：每回合第一次使用斬時傷害 +1
- 趙雲技能「龍膽」：閃可當斬、斬可當閃
- 諸葛亮技能「觀星」：每回合開始時先查看牌堆頂最多 3 張，選 1 張加入手牌，其餘放回牌堆底，再正常抽 2 張牌
- 武將定位：關羽「攻擊爆發」、趙雲「攻防靈活」、諸葛亮「策略控牌」
- 第一章：黃巾亂起，8 關流程
- 關卡敵人池系統：第 1～7 關依各關敵人池隨機選出，第 8 關固定呂布
- Mini-boss：張梁、張寶
- 最終 Boss：第 8 關虎牢關前挑戰呂布
- 玩家手牌與士氣系統
- 牌組總數 31 張
- 基礎卡牌：斬、閃、酒、兵書、破甲
- 戰術卡牌：連斬、固守、激勵、火攻
- 裝備卡系統：裝備後效果持續到整局結束
- 三張裝備卡：青龍偃月刀、的盧馬、太平要術
- 事件關卡系統：擊敗第 1 關或第 2 關後，有 50% 機率先遇到事件，再進入戰後獎勵
- 三個事件：荒村補給、軍師獻策、伏兵突襲
- 路線選擇系統：戰後獎勵後選擇下一關路線，決定敵人體力與報酬
- 三條路線：山道、官道、險道
- 敵人行動：普通攻擊、猛攻、防守、蓄力
- 敵人攻擊時可選擇使用閃或承受傷害
- 戰後獎勵三選一與玩家成長
- 戰鬥紀錄、關卡提示、操作回饋與輕量動畫
- 首頁教學、目前版本特色、遊戲內提示與快速規則

- 第一批 TTS 語音：章節開場旁白、三位武將登場與呂布登場
- 開頭動畫：首頁可觀看或略過 9:16 第一章開場動畫
- 手機遊玩 UI / UX 優化：戰鬥 HUD、底部手牌操作區、戰鬥紀錄與狀態設定收合
- 首頁與戰鬥視覺修正：戰鬥角色圖 / 敵人圖回到主畫面，首頁武將選擇加入音效與登場語音試聽

## 本地啟動

安裝依賴：

```sh
npm install
```

啟動開發伺服器：

```sh
npm run dev
```

開啟 [http://localhost:3000](http://localhost:3000)。

## 視覺資產系統

v0.10.0 先建立視覺資料欄位與 UI 顯示位置，尚未放入正式圖片。資料層已為武將、敵人、關卡、事件、路線與卡牌加入 placeholder key 與 `visualPrompt`，方便後續接入 AI 生成圖像。

- 武將：`portrait`、`avatar`、`visualPrompt`
- 敵人：`portrait`、`visualPrompt`
- 關卡：`backgroundImage`、`visualPrompt`
- 事件：`image`、`visualPrompt`
- 路線：`image`、`visualPrompt`
- 卡牌：`illustration`、`visualPrompt`

`visualPrompt` 用來記錄未來生成圖像時的方向，例如角色立繪、敵人圖、關卡背景、事件圖、路線圖與卡牌插圖。placeholder 採用本地 key 與漸層圖塊呈現，避免正式圖片尚未產出時造成 build 或部署問題。

下一步會使用 AI 生成第一批角色、敵人與背景圖片，逐步替換目前的 placeholder。

## 人物台詞系統

v0.11.0 新增第一版文字台詞系統，先不加入音檔、不串接 TTS 或外部 API。台詞資料以 `DialogueLine` 管理，包含說話者、觸發時機、台詞內容、語氣、立繪與預留的 `audioKey`。

目前支援：

- 武將台詞：關羽、趙雲、諸葛亮的登場、開戰、攻擊、受傷、低血量與勝利台詞。
- 敵人登場台詞：黃巾兵、山賊頭目、西涼騎兵與 Boss 呂布。
- 章節旁白：第一章開場敘事。
- 關卡旁白：荒村初戰與虎牢關前。
- 通關 / 戰敗旁白：結果頁與戰鬥結束時可使用同一組台詞資料。

遊戲頁新增「台詞框」，顯示最新重要台詞、說話者名稱與說話者類型。`audioKey` 目前只作為未來接 TTS 或真人配音音檔的索引，不會載入或播放任何音訊。

## 音效系統

v0.12.0 新增第一版音效系統，先不加入真實音檔、不使用外部 API，也不引入大型音效套件。遊戲使用瀏覽器 Web Audio API 生成簡單提示音，提供出牌、攻擊、閃避、受傷、回復、抽牌、獎勵、事件、路線、Boss 登場、勝利與戰敗等基本回饋。

音效預設關閉，避免瀏覽器自動播放限制與干擾展示環境。玩家可在遊戲頁手動切換「音效：關 / 音效：開」，設定會以 localStorage 儲存在本機；若瀏覽器或 SSR / 測試環境沒有 `window`、`AudioContext` 或 localStorage，音效工具會安全 no-op，不影響遊戲流程。

下一步可接入真實音效素材，或搭配既有台詞 `audioKey` 延伸到 TTS / 角色語音。

## TTS 配音規劃

v0.12.1 建立第一版 TTS 配音素材清單，文件位於 [`docs/tts-dialogue-assets-v1.md`](docs/tts-dialogue-assets-v1.md)，結構化資料位於 `lib/game/ttsManifest.ts` 的 `TTS_DIALOGUE_MANIFEST`。

目前只整理配音規劃，不加入真實語音檔、不播放語音，也不串接外部 TTS API。`audioKey` 是 `DialogueLine` 與 TTS manifest 的對應索引，用來把台詞、角色聲線、建議語速、用途與未來音檔路徑連在一起。

已建立音檔資料夾規劃：

- `public/audio/voices/guan-yu`
- `public/audio/voices/zhao-yun`
- `public/audio/voices/zhuge-liang`
- `public/audio/voices/lu-bu`
- `public/audio/voices/enemies`
- `public/audio/narration`

下一步可使用 TTS 工具依照 manifest 生成語音，輸出 MP3 後放入 `public/audio` 對應資料夾，再由遊戲根據 `audioKey` 載入。

## 語音播放框架

v0.12.2 建立第一版角色語音 / 旁白語音播放框架。遊戲頁新增「語音：關 / 語音：開」設定，預設關閉並可用 localStorage 保存。台詞框更新時，若 `DialogueLine.audioKey` 存在且語音開關已開啟，會透過 `lib/game/voice.ts` 查詢 `TTS_DIALOGUE_MANIFEST` 並嘗試播放對應音檔。

v0.12.3 已導入第一批 5 筆 MP3 語音：

- 章節開場旁白：`/audio/narration/chapter-1-intro.mp3`
- 關羽登場：`/audio/voices/guan-yu/guan-yu-intro.mp3`
- 趙雲登場：`/audio/voices/zhao-yun/zhao-yun-intro.mp3`
- 諸葛亮登場：`/audio/voices/zhuge-liang/zhuge-liang-intro.mp3`
- 呂布登場：`/audio/voices/lu-bu/lu-bu-intro.mp3`

語音預設仍為關閉，玩家可在遊戲頁手動開啟。其他台詞仍為 `planned`，尚未有音檔，不會播放。若音檔缺失、瀏覽器阻擋或播放失敗，語音框架會安全 no-op，不會中斷遊戲或影響既有 Web Audio 音效系統。

TTS manifest 狀態定義：

- `planned`：已規劃，尚未生成。
- `generated`：已生成但尚未導入專案。
- `ready`：音檔已放入 `public/audio` 且可播放。

未來把其他 MP3 放入 `public/audio` 對應路徑，並將 manifest status 改為 `ready` 後，語音播放框架即可根據 `audioKey` 播放更多角色語音或旁白語音。

## 開頭動畫

v0.13.0 導入第一章 9:16 直式開頭動畫，影片位置為 `public/videos/opening-intro.mp4`，對應 public path 為 `/videos/opening-intro.mp4`。首頁提供「觀看開場動畫」與「略過動畫，直接開始」兩種路徑；動畫播放完畢後會顯示「開始遊戲」，並沿用玩家已選擇的武將。

本版使用 HTML video，不引入大型影片播放器套件。影片預設靜音並保留播放器控制，避免瀏覽器自動播放限制；若影片載入失敗，首頁會顯示 fallback 文案，玩家仍可直接開始遊戲。

開場動畫由 AI 生成首幀圖與影片工具製作，並配合 Suno Studio 生成 20 秒循環音樂，用於呈現第一章：黃巾亂起的開場氣氛。

## 手機遊玩 UI / UX

v0.13.1 針對手機直屏遊玩節奏做優化，目標是降低戰鬥中反覆上下滑動的頻率，同時保留桌機版完整資訊布局。

- 手機戰鬥 HUD：精簡顯示敵人名稱、類型、HP、狀態，以及武將名稱、HP、目前狀態。
- 底部手牌操作區：手機版手牌固定在底部並可橫向滑動，結束回合與回應敵人攻擊按鈕靠近主要操作區。
- 戰鬥紀錄收合：手機版預設只顯示最近紀錄，可展開查看最近 8 條。
- 狀態與設定收合：手機版將目前強化、已裝備、音效、語音、快速規則與戰局資訊收進「狀態與設定」。
- 事件 / 獎勵 / 路線選擇優化：手機版選項維持垂直排列與清楚標題，避免在說明與選項間來回滑動。
- 首頁行動版流程優化：手機版降低首頁主視覺高度，將武將選擇與開始遊戲放在開場動畫之前，保留觀看動畫作為次要路徑。

v0.13.2 修正首頁互動與戰鬥視覺：

- 戰鬥畫面手機 HUD 加回武將與敵人縮圖；桌機版仍保留完整角色 / 敵人圖片面板。
- 已導入圖片的武將與敵人會顯示正式圖片，未導入圖片的敵人仍使用 placeholder fallback。
- 首頁點選武將會播放選擇音效；若「角色語音」開啟，會播放該武將登場語音。
- 開場動畫入口移到首頁上方，預設只顯示精簡入口，點擊後才展開播放器。
- 「怎麼玩？」與「目前版本特色」改為收合區塊，縮短手機首頁長度。

v0.13.3 優化開場動畫觀看體驗：

## 第一章：黃巾亂起

亂世初起，黃巾餘亂四起，英雄踏上第一段征途。第一章共 8 關：

1. 荒村初戰：荒村煙塵未散，黃巾餘黨正在掠奪糧草。敵人池：黃巾兵、黃巾弓手。
2. 山道伏兵：山道狹窄，伏兵隱於林間。敵人池：黃巾弓手、黃巾力士。
3. 破廟夜戰：破廟燈火搖曳，敵影在夜色中逼近。event-heavy 關卡，敵人池：黃巾兵、山賊頭目。
4. 黑山賊寨：黑山賊寨盤踞山間，守備森嚴。敵人池：山賊頭目、黑山賊將。
5. 西涼突騎：馬蹄聲急，西涼騎兵席捲而來。敵人池：西涼騎兵、黑山賊將。
6. 古戰場遺跡：古戰場埋藏著舊日兵戈，也藏著未知危機。event-heavy 關卡，敵人池：黃巾力士、西涼騎兵。
7. 黃巾祭壇：祭壇妖風四起，黃巾殘部正進行詭異儀式。Mini-boss 敵人池：張梁、張寶。
8. 虎牢關前：虎牢關前，赤兔嘶鳴，呂布橫戟而立。最終 Boss：呂布。

event-heavy 關卡勝利後事件觸發機率為 75%；其他非最終關卡維持 50%。第 8 關擊敗呂布後不觸發事件、不進入獎勵與路線選擇，直接完成第一章。

## 戰術卡牌系統

v0.8.0 將玩家牌組擴充為 31 張，新增 4 種戰術卡，每種 2 張。連斬與火攻不視為普通「斬」，不觸發關羽武聖、斬傷害強化或青龍偃月刀。

目前牌組構成：斬 x8、閃 x5、酒 x3、兵書 x2、破甲 x2、青龍偃月刀 x1、的盧馬 x1、太平要術 x1、連斬 x2、固守 x2、激勵 x2、火攻 x2。

- 連斬：攻擊牌。對敵人造成 1 點傷害；若使用前敵人已不是滿血，使用後抽 1 張牌。
- 固守：防禦牌。本回合下一次受到傷害 -1，最低為 0；觸發後消失，進入下一回合或下一關也會重置。
- 激勵：策略牌。抽 1 張牌，並回復 1 點體力，不可超過最大體力。
- 火攻：策略 / 攻擊牌。敵人處於蓄力狀態時造成 2 點傷害並打斷蓄力；否則造成 1 點傷害。

## 裝備系統

裝備卡使用後會移出手牌並進入「已裝備」區，效果持續到整局遊戲結束。每種裝備只能裝備一張。

- 青龍偃月刀：攻擊流。每回合第一次使用斬時額外 +1 傷害。
- 的盧馬：防守流。每場戰鬥第一次受到敵人攻擊時自動閃避一次，進入下一關後重置。
- 太平要術：策略流。每回合第一次使用兵書時額外抽 1 張牌。

## 事件系統

擊敗第 1～7 關敵人後，可能進入事件階段。事件處理完成後才會進入戰後獎勵。一般關卡事件觸發機率為 50%，event-heavy 關卡為 75%。第 8 關呂布被擊敗後不觸發事件，直接通關。

- 荒村補給：補給事件。選擇休整後回復 2 點體力，不可超過最大體力。
- 軍師獻策：策略事件。選擇聽取建議後抽 2 張牌。
- 伏兵突襲：危險事件。選擇突圍後失去 1 點體力，若未戰敗則獲得斬傷害 +1 強化。

## 路線選擇系統

擊敗第 1～7 關敵人後，流程為「可能事件 → 戰後獎勵 → 路線選擇 → 下一關」。路線只影響下一場戰鬥。

- 山道：低風險。下一關敵人最大體力與目前體力 -1，最低不低於 1。
- 官道：中風險。下一關敵人維持正常難度。
- 險道：高風險。下一關敵人最大體力與目前體力 +2；若成功擊敗該關敵人，該次戰後獎勵改為 4 選 1，最多不超過全部可用獎勵數量。

## 敵人池系統

第 1～7 關敵人會從各關對應敵人池中隨機選出，第 8 關 Boss 固定為呂布。路線效果會套用到下一關選出的敵人身上。

第一章敵人池重點：

- 黃巾兵、黃巾弓手、黃巾力士：第一章前段主要敵人。
- 山賊頭目、黑山賊將、西涼騎兵：中段精英敵人。
- 張梁：黃巾三公將軍之一，攻勢猛烈，擅長蓄力後爆發。
- 張寶：擅用妖術迷惑敵軍，防守與蓄力能力較強。
- 呂布：第 8 關最終 Boss，虎牢關前的最終考驗。
