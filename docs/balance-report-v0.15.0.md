## 初步觀察

- 目前整體勝率為 92.7%，可作為後續數值調整基準。
- 勝率最低武將為 諸葛亮（78%）。

- basic-safe-strategy 最常選擇 險道（923 次）。

## 建議調整方向

- 先用更高局數重跑模擬，確認勝率是否穩定。
- 若某武將勝率顯著低於其他武將，再檢查其初始手牌、技能觸發頻率與續航能力。
- 若死亡集中於單一關卡，優先檢查該關敵人池與行動牌組。
- 整體勝率偏高，可考慮提高第 7～8 關壓力，或降低高報酬路線收益。
