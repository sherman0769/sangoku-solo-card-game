# 第一批 AI 圖像資產清單

本文件整理《三國單騎傳》第一批 AI 圖像資產需求。此階段只規劃圖片用途、建議路徑與生成提示詞，不放入正式圖片。

## 統一美術風格

風格方向：

「東方史詩卡牌風 × 三國亂世 × 半寫實插畫 × 遊戲概念美術」

風格要求：

- 不使用現有影視、遊戲、動漫角色形象
- 不模仿任何特定 IP
- 三國氛圍
- 電影感光影
- 精緻卡牌遊戲質感
- 可用於教學展示
- 避免血腥暴力
- 不要文字、水印、Logo
